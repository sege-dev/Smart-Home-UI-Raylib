# 🏠 OmniGuard Premium - Smart Home Dashboard

**OmniGuard** is a high-performance, aesthetically pleasing Smart Home Control Center built entirely in **C** using the **Raylib** library. It features a modern **Glassmorphism** design, real-time atmosphere simulation, and interactive device controls.

---

## ✨ Key Features

- **Modern Glassmorphism UI:** Semi-transparent panels with soft borders and dynamic depth effects for a premium look.
- **Dynamic Atmosphere Simulation:** Real-time climate monitoring. The room's visual atmosphere (tint) changes dynamically based on current temperature (Warm/Cold).
- **Interactive Mouse Controls:** Fully interactive dashboard. Toggle smart lighting and monitor system status with simple clicks.
- **Seamless 2D Integration:** High-resolution assets integrated into a custom-built graphics engine for maximum performance and clarity.
- **Unified Visual Experience:** A cohesive interface where the UI blends perfectly with the room's background for an immersive feel.

## 🛠️ Tech Stack

- **Language:** C (C99 Standard)
- **Graphics Library:** [Raylib](https://www.raylib.com/)
- **UI Design Style:** Glassmorphism / Dark Mode
- **Build System:** Makefile / MinGW-w64

---

## ⚙️ Installation & Build

1. **Clone the repository:**
   `git clone https://github.com/sege-dev/Smart-Home-UI-Raylib.git`

2. **Build the project:**
   `mingw32-make`

3. **Run the application:**
   `./gui_test.exe`

---

## 🎮 Controls

- **Mouse Interaction:** Click on Dashboard Cards to toggle devices (e.g. Smart Lighting)
- **Keyboard Simulation (Arrows):** Change Temperature Manually
- **Keyboard Shortcut ('L'):** Toggle Lighting
